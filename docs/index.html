Features

Auto Offload

Large outputs (>8KB) automatically saved to disk. Smart preview with first/last 10 lines stays in context.

Privacy First

Only logs tool names and parameter keys. Never logs your actual data, secrets, or command arguments.

Auto Cleanup

Stale outputs cleaned on session start. No manual maintenance needed.

MCP Aware

Intercepts MCP tools, clamps pagination to prevent runaway queries. Your browser automation stays efficient.
